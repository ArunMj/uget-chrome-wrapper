Introduction

We integrate the best download manager with the best web browser.

uGet

uGet is an Open Source download manager application for GNU/Linux developed with GTK+, which also comes packaged as a portable Windows application.

Google Chrome, Chromium, Vivaldi & Opera

The fast, secure, and free web browsers built for the modern web. Opera addon is not approved yet. For the moment, Opera users can install the addon using Download Chrome Extension.

Linux & Windows

This project is an attempt to fill the gap in download accelerators available for Linux. However, setup for Windows is available for people using uGet in Windows.
